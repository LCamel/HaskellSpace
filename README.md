# HaskellSpace

這是一個隨開即用的 Haskell 環境. 建構於 [GitHub Codespaces](https://github.com/features/codespaces) 之上, 每個月可以免費使用 60 小時.

(影片)


## Stack

如果需要使用 stack, 請執行:
```
stack new foo --bare --resolver "$(cat ~/stackage-version)"
```


## 進階使用

如果不想受限於網頁介面或 Codespaces 的使用時間上限, 請參考本專案使用的 [LCamel/haskell-devcontainer](https://github.com/LCamel/haskell-devcontainer)
